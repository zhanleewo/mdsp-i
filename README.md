**感谢张颢教授。**

我并非张老师门下的学生，但他的随机过程、数字信号处理等视频课程，让我感受到了一位学者对教学朴素而执着的诚意。

张老师的课上，最让我印象深刻的是那满墙工整的板书——七八块黑板写了又擦、擦了又写，公式推导不急不缓，仿佛不是在“讲课”，而是在带着你一起“想问题”。他的表述极为精确，每一句话都有明确的指涉，每一个符号都被准确地唤出名字，每一步推导的逻辑衔接都清晰可辨。这种精确不是刻意的雕琢，而是长期思考锤炼出的语言习惯。

他贯穿始终的三句话——**No data, no truth. No analytics, no understanding. No programming, no cognition.**——提醒每一个学习信号处理的人：理论必须扎根于数据，直觉必须经得起推导，思想必须能跑成代码。而每当内容超出学生当前的理解边界时，他也会温和地提醒一句：“大家要有一定的数学成熟度，不需要每一个证明都亲自完成。”没有居高临下的说教，只有过来人对初学者的体谅。

在讲完LPC前向后向预测算法之后，他在黑板边安静地写下一首诗：

> 手把青秧插满田，低头便见水中天。  
> 心地清净方为道，退步原来是向前。

前向预测推未来，后向预测将序列反转——算法在反转中获得了更对称的结构。“退步原来是向前”，既是对算法的总结，也像是对学习本身的一种提醒。

虽未亲临课堂，但一个未曾谋面的学习者，能通过视频完成这样一次思想跟随，已是难得的运气。

侯强教授是把我领进这个领域的人。那时我刚开始学机器学习，没有方向，在QQ上到处找教学群，大多只面向自己的学生。侯老师让我进了群，还根据我的情况，不厌其烦地给我推荐各种学习材料。后来回想，那时我大概正处在学习曲线的最低谷——侯老师的指引让我慢慢找到了方向感。

侯老师的风格偏几何，和张老师偏代数的视角恰好互补。他总能用最干练的语言说出事情的本质。记得聊到Wasserstein距离时，他告诉我最优传输理论的奠基人叫Monge，巴黎有一个地铁站就叫Monge，就是为了纪念他。从那以后，每当想到最优传输，脑子里就会浮现一个地铁站的名字——学问和历史就这样连在了一起。

也有些时候，是我歪打正着发现一些材料，请他帮我把关。我给他推荐过Wainwright的《High-Dimensional Statistics》，他对那本书爱不释手。我自己读着还非常吃力，转而去看马毅老师的书。正是在那本书里，我遇到了张老师课上隆重介绍过的Candes关于稀疏恢复的下界证明——两边学到的知识奇妙地交汇在了一起。

他于我而言，不像一位在群里指点迷津的师长，更像是站在路口给你指一下路的人——不拉你走，但告诉你该往哪看。没有他，我大概还在某个QQ群里到处问“这到底该怎么学”。

这些指点，我一直记得。

**学习动机：从机器学习回溯信号处理**

在学习机器学习和神经网络的过程中，我读了不少论文，也看了大量网络架构，却始终有一种说不清的隔阂感——我不明白为什么要这样设计网络，设计这些架构的人，他们的理论根据到底是什么。我能复现代码，却无法复现思想。

摸索了很久，隐约感觉到答案也许藏在随机过程里。于是找到了张老师的随机过程课程，也顺带发现了他的数字信号处理系列。当我学完这些课程之后，某些长久以来悬而未决的困惑，开始一块一块地落地：

YOLO那些锚框的设计，根源或许就是匹配滤波；DeepSORT的目标追踪，本质上就是参数化的卡尔曼滤波；而Transformer里最核心的attention score，当你在随机过程的语境下看到它时，会猛然意识到——那不过就是在算全局的相关性函数。那一刻我才真正理解，为什么attention的上限要比CNN高，因为CNN的卷积核是局部的、固定的，而相关性函数是在整个空间上搜索相似性，这是两种完全不同的信息提取逻辑。

后来跟着张老师的推导思路，我也徒手推到了DDPM；在SVM那一讲中，更是第一次真正领悟了对偶的本质——内层与外层的嵌套关系，原来可以用这样清晰的逻辑拆解出来。

这些“开悟”的时刻，与其说是灵光一现，不如说是张老师的课程帮我补齐了缺失的那一块拼图：机器学习的许多设计，本质上都是信号处理的思路在不同问题上的投影。理解了信号处理，再看机器学习，就像从局部视图切换到全局地图。

**关于这份笔记**

我希望这份笔记并不是要取代你亲自动手推导的过程——张老师的课，真正的收获从来都在你跟着他一行一行写下来的那个时刻。这份笔记的价值，在于课后复习时，能提供足够丰富的文字说明，帮你回忆起当初推导的逻辑脉络、用到的关键技巧，以及那些在板书的缝隙里一闪而过的洞察。笔记是路标，路还得自己走。

这是一套自顶向下、从理论到算法的**现代数字信号处理（Modern Digital Signal Processing）**课程笔记，共 19 讲，分为三个单元。全书以统计信号处理为核心视角，将线性代数、概率论与优化理论编织成一条连贯的主线——不再把 DSP 视为一组孤立的滤波器设计技巧，而是看作**在不确定性中做出最优推断**的数学框架。

**三条格言贯穿始终：**

> **No data, no truth.** — 一切推断必须根植于观测数据。
>
> **No analytics, no understanding.** — 直觉必须经得起数学的推敲。
>
> **No programming, no cognition.** — 理论需落地为可运行的代码。

### 内容结构

全书沿 **"估计什么 → 多好算好 → 怎么估计 → 如何高效求解"** 的主线展开：

- **第一单元（01–09）**：建立统计信号处理的理论基石。从概率论复习出发，引入最小方差无偏估计（MVUE）、Cramér-Rao 下界（CRLB），进而覆盖维纳滤波、卡尔曼滤波、线性预测编码（LPC）等经典最优滤波方法，最后以正则化与 SVM 收尾，连接机器学习视角。
- **第二单元（10–14）**：聚焦自适应滤波器，处理**时间耦合**这一核心矛盾。从 LMS 的随机梯度思想出发，逐步推导 RLS、QR-RLS 等精确最小二乘方法，并引入 PCA/TLS 进行信号子空间分析，最终在 NLMS 与仿射投影算法（APA）中达到计算量与收敛速度的工程平衡。
- **第三单元（15–19）**：深入谱分析，解决**"从有限数据估计连续功率谱"**这一根本难题。从经典非参数方法（周期图、Bartlett、Welch）出发，引入长球波函数（PSWF）实现最优窗口设计，再过渡到 ARMA 参数化建模与最大熵谱估计，最后以 Kolmogorov-Szegő 理论给出随机过程线性预测的数学极限。

## 先修要求

- **微积分**：熟练掌握求导、积分、多元微分与级数展开，是理解 CRLB 推导、梯度优化和谱分析中渐近理论的基本工具。
- **线性代数**：矩阵运算、特征值分解、二次型与投影理论是全书的核心数学语言——从 Wiener-Hopf 方程到 Kalman 滤波的状态空间模型，无不在线性代数的框架内运作。
- **概率论**：需达到"本能反应"的熟练度——看到随机变量能条件反射地想到分布、期望与方差；看到两个变量能自然推导联合、边缘与条件分布。Bayes 公式更是贯穿全书的灵魂。
- **复变函数**（少量涉及）：了解 Cauchy 积分定理与留数计算，对谱分析中 Kolmogorov-Szegő 公式的推导会有更通透的理解。
- **信号与系统 / 经典 DSP**（推荐）：若有背景，对滤波、采样、频域分析等章节会有更直观的物理直觉。

## 参考书目

本讲义是以下经典著作核心思想的提炼与串联：

1. **《统计信号处理基础》**（Steven M. Kay）—— 估计与检测理论领域无可争议的圣经。本书的估计理论框架（MVUE、CRLB、充分统计量）直接来源于此。
2. **《自适应滤波器》**（Simon Haykin）—— 将统计信号处理从理论推向工程实现的关键著作。维纳滤波、LMS、RLS 等章节均以此为主要参考。
3. **《现代信号谱分析》**（Stoica & Moses）—— 从周期图到现代参数化谱估计的完整图谱，第三单元的核心方法论来源。
4. **《概率机器学习：一种可引入的方法》**（Kevin P. Murphy）—— 作为 Bayes 视角的补充读物，为正则化与核方法提供更广阔的上下文。
