**感谢张颢教授。**

我并非张老师门下的学生，未曾坐在他的教室里上过一节课。但通过他的随机过程、数字信号处理、现代数字信号处理I和II这些视频课程，我依然感受到了一个学者对教学最朴素也最执着的诚意。

张老师的课堂，始于满墙工整到近乎严苛的板书——一节课下来，七八块乃至十来块黑板写了又擦、擦了又写，公式推导不急不缓，仿佛不是在“讲课”，而是在带着你一起“想问题”。这种对板书秩序的坚持，与他推崇的小平邦彦对笔记的重视一脉相承——真正的理解，从来都来自亲手写下的过程，而不是屏幕前一闪而过的PPT。

而他的声音，同样让这种“跟随”变得格外从容——语速平缓，字正腔圆，像极了央视新闻主播的发音，每一句话都清清楚楚地送到耳朵里。在推导最密集、符号最纷乱的时刻，这种稳定的节奏反而成了一种锚点：你不必费力辨认他在说什么，只需把全部注意力放在“他在推什么”上。慢，却恰好给了思考以空间。

更难得的是他课堂表述的精确。听过许多课的人大概都有过这样的体验：老师口中满是“这个”“那个”，手指在黑板上虚点，你拼命追踪他的指尖，却怎么也跟不上他到底指的是哪一项。张老师从来没有这个问题——他的每一句话都有明确的指涉，每一个符号都被准确地唤出名字，每一步推导的逻辑衔接都清晰可辨。这种精确不是刻意的雕琢，而是长期思考锤炼出的语言习惯：想得有多清楚，说出来就有多清楚。

张老师最令人叹服的地方，在于他能把泛函分析、变分法这样高深的数学工具，讲到只具备基础微积分和线性代数知识的学生都能听懂——概念的骨架被拆解成一步步可以跟随的推演，抽象的数学在具体的信号处理问题中落地生根。而每当内容确实超出学生当前的理解边界时，他也会停下来，温和地安慰一句：“大家要有一定的数学成熟度，不需要每一个证明都亲自完成。”这句话里没有居高临下的说教，只有一份过来人对初学者的体谅——有些门槛，时间到了自然会迈过去，不必在当下苛责自己。

课上那些看似信手拈来的“金句”，同样让人回味。他讲“人生是泊松过程”，说我们的人生很多时候都在等待一个机遇的出现。而贯穿始终的三句话——**No data, no truth. No analytics, no understanding. No programming, no cognition.** ——则像一把尺子，提醒每一个学习信号处理的人：理论必须扎根于数据，直觉必须经得起推导，思想必须能跑成代码。

张老师在推导完LPC的前向后向预测算法之后，临近那堂课的尾声，在黑板右边安静地写下一首诗：

> 手把青秧插满田，低头便见水中天。  
> 心地清净方为道，退步原来是向前。

那一刻，整门课程从繁复的公式推导回归到一种极简的智慧：前向预测是依据过去推未来，后向预测的核心则是将序列反转——时间的方向被倒转，算法却在反转之中获得了更对称、更优雅的结构。而“退步原来是向前”恰好映照了这种反转的深意：有时候，向后看，恰恰是为了更好地向前走。这句话既是对算法的总结，也是对学习本身最温柔的注脚。

感谢张老师把一门硬核课程讲出了温度与深度，也感谢这系列视频让一个未曾谋面的学习者，能够跨越屏幕完成一次完整的思想跟随。虽未亲临课堂，但受益同样真切。

同时，也要感谢中国地质大学侯强教授。在我最开始学机器学习的时候，没有方向，四处乱撞。在QQ上找教学群时，大多仅限于授课对象，我也理解。侯老师很包容，不但收留了我，还从不嫌弃我的无知，不厌其烦地为我推荐各种学习材料。根据学习的开悟曲线，我当时正处在绝望之谷的最低点——侯老师的鼓励和指引，让我从那里一步步走了出来，逐渐有了方向感。

这两年在群里，侯老师和我以及和其他学生的每一次交流，都是我查漏补缺的宝贵材料。我并非侯老师门下的学生，但他从未把我当作门外人看待。每次向侯老师请教，他都能用最干练的语言说出事情的本质——三言两语就把一个复杂概念的核心拎出来，绝不拖泥带水。侯老师的视角偏几何，恰好与张老师偏代数的风格形成互补。

在侯老师的引领下，我得以窥见一个更广阔的世界——这些年来他带我“入门”的领域，不胜枚举：流形、深度展开、梯度流、最优传输理论、VAE、变分学，还有优化中的oracle概念，以及自适应滤波与神经网络之间的深刻关联，都源于他的点拨。也正是了解了拓扑和流形我才真正理解了为什么transformer的embedding层叫embedding层。当我和侯老师聊到张老师压缩感知课程中0范数到1范数的松弛条件是 \( n > m \log N \) 时，侯老师带我了解了随机矩阵，还给我推荐了华中科技大学廖振宇老师的《Random Matrix Methods for Machine Learning》；聊到随机过程时，我找到了Dobrow的《Introduction to Stochastic Processes with R》，侯老师也给予了认可和补充；聊到仿真和采样时，他推荐了朱松纯老师的《Monte Carlo Methods in Vision, Graphics and Learning》；聊到凸优化时，他推荐了尤里·涅斯捷罗夫的《凸优化教程》；聊到Wasserstein距离时，他介绍了雷娜的《Optimal Transportation Theory and Computation》，还特意告诉我奠基人叫Monge，巴黎有一个地铁站就叫Monge，就是为了纪念他——这一点我一直记得很深，因为从那以后，每当我想到最优传输，脑子里就会浮现一个地铁站的名字，学问和历史就这样连在了一起。

也有时候，是我歪打正着发现一些材料，侯老师帮我确认价值。比如我在看高维统计时，给侯老师推荐了Martin J. Wainwright的《High-Dimensional Statistics: A Non-Asymptotic Viewpoint》，侯老师对这本书爱不释手。我自己读着还非常吃力，便转而看马毅老师的《基于低维模型的高维数据分析》。正是在这本书里，我遇见了张老师在课上隆重介绍过的一句话——“这是工科同学唯一能看懂的一段Fields级别数学解析”——Candes关于稀疏恢复中0范数松弛成1范数的下界证明。那一刻，张老师课堂上提到的理论，在马老师的书中落到了可以逐字读懂的纸面上，而这本书的方向也得到了侯老师的认可。三个人，三门课，三本书，在我的学习路径上奇妙地交汇在了一起。

但扫盲只是一部分，更关键的是，侯老师在我最迷茫的时候给了我方向感——告诉我该往哪里走、该读什么、该补什么。凡此种种，这几年下来，侯老师带我入的门，早已数不清了。他于我而言，早已不止是一位在QQ群里指点迷津的师长，更像一位站在路口替我点灯引路的人。没有他，我不知道现在还在哪片迷雾里打转。这份恩情，我一直记在心里。

**学习动机：从机器学习回溯信号处理**

在学习机器学习和神经网络的过程中，我读了不少论文，也看了大量网络架构，却始终有一种说不清的隔阂感——我不明白为什么要这样设计网络，设计这些架构的人，他们的理论根据到底是什么。我能复现代码，却无法复现思想。

摸索了很久，隐约感觉到答案也许藏在随机过程里。于是找到了张老师的随机过程课程，也顺带发现了他的数字信号处理系列。当我学完这些课程之后，某些长久以来悬而未决的困惑，开始一块一块地落地：

YOLO那些锚框的设计，根源或许就是匹配滤波；DeepSORT的目标追踪，本质上就是参数化的卡尔曼滤波；而Transformer里最核心的attention score，当你在随机过程的语境下看到它时，会猛然意识到——那不过就是在算全局的相关性函数。那一刻我才真正理解，为什么attention的上限要比CNN高，因为CNN的卷积核是局部的、固定的，而相关性函数是在整个空间上搜索相似性，这是两种完全不同的信息提取逻辑。

后来跟着张老师的推导思路，我也徒手推到了DDPM；在SVM那一讲中，更是第一次真正领悟了对偶的本质——内层与外层的嵌套关系，原来可以用这样清晰的逻辑拆解出来。

这些“开悟”的时刻，与其说是灵光一现，不如说是张老师的课程帮我补齐了缺失的那一块拼图：机器学习的许多设计，本质上都是信号处理的思路在不同问题上的投影。理解了信号处理，再看机器学习，就像从局部视图切换到全局地图。

**关于这份笔记**

我希望这份笔记并不是要取代你亲自动手推导的过程——张老师的课，真正的收获从来都在你跟着他一行一行写下来的那个时刻。这份笔记的价值，在于课后复习时，能提供足够丰富的文字说明，帮你回忆起当初推导的逻辑脉络、用到的关键技巧，以及那些在板书的缝隙里一闪而过的洞察。笔记是路标，路还得自己走。

这是一套自顶向下、从理论到算法的**现代数字信号处理（Modern Digital Signal Processing）**课程笔记，共 19 讲，分为三个单元。全书以统计信号处理为核心视角，将线性代数、概率论与优化理论编织成一条连贯的主线——不再把 DSP 视为一组孤立的滤波器设计技巧，而是看作**在不确定性中做出最优推断**的数学框架。

**三条格言贯穿始终：**

> **No data, no truth.** — 一切推断必须根植于观测数据。
>
> **No analytics, no understanding.** — 直觉必须经得起数学的推敲。
>
> **No programming, no cognition.** — 理论需落地为可运行的代码。

### 内容结构

全书沿 **"估计什么 → 多好算好 → 怎么估计 → 如何高效求解"** 的主线展开：

- **第一单元（01–09）**：建立统计信号处理的理论基石。从概率论复习出发，引入最小方差无偏估计（MVUE）、Cramér-Rao 下界（CRLB），进而覆盖维纳滤波、卡尔曼滤波、线性预测编码（LPC）等经典最优滤波方法，最后以正则化与 SVM 收尾，连接机器学习视角。
- **第二单元（10–14）**：聚焦自适应滤波器，处理**时间耦合**这一核心矛盾。从 LMS 的随机梯度思想出发，逐步推导 RLS、QR-RLS 等精确最小二乘方法，并引入 PCA/TLS 进行信号子空间分析，最终在 NLMS 与仿射投影算法（APA）中达到计算量与收敛速度的工程平衡。
- **第三单元（15–19）**：深入谱分析，解决**"从有限数据估计连续功率谱"**这一根本难题。从经典非参数方法（周期图、Bartlett、Welch）出发，引入长球波函数（PSWF）实现最优窗口设计，再过渡到 ARMA 参数化建模与最大熵谱估计，最后以 Kolmogorov-Szegő 理论给出随机过程线性预测的数学极限。

## 先修要求

- **微积分**：熟练掌握求导、积分、多元微分与级数展开，是理解 CRLB 推导、梯度优化和谱分析中渐近理论的基本工具。
- **线性代数**：矩阵运算、特征值分解、二次型与投影理论是全书的核心数学语言——从 Wiener-Hopf 方程到 Kalman 滤波的状态空间模型，无不在线性代数的框架内运作。
- **概率论**：需达到"本能反应"的熟练度——看到随机变量能条件反射地想到分布、期望与方差；看到两个变量能自然推导联合、边缘与条件分布。Bayes 公式更是贯穿全书的灵魂。
- **复变函数**（少量涉及）：了解 Cauchy 积分定理与留数计算，对谱分析中 Kolmogorov-Szegő 公式的推导会有更通透的理解。
- **信号与系统 / 经典 DSP**（推荐）：若有背景，对滤波、采样、频域分析等章节会有更直观的物理直觉。

## 参考书目

本讲义是以下经典著作核心思想的提炼与串联：

1. **《统计信号处理基础》**（Steven M. Kay）—— 估计与检测理论领域无可争议的圣经。本书的估计理论框架（MVUE、CRLB、充分统计量）直接来源于此。
2. **《自适应滤波器》**（Simon Haykin）—— 将统计信号处理从理论推向工程实现的关键著作。维纳滤波、LMS、RLS 等章节均以此为主要参考。
3. **《现代信号谱分析》**（Stoica & Moses）—— 从周期图到现代参数化谱估计的完整图谱，第三单元的核心方法论来源。
4. **《概率机器学习：一种可引入的方法》**（Kevin P. Murphy）—— 作为 Bayes 视角的补充读物，为正则化与核方法提供更广阔的上下文。
